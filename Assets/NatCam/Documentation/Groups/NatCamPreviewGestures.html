NatCamPreviewGestures

class in NatCamU / inherits from NatCamPreviewBehaviour

Description

This component is useful for tracking mouse and touch gestures for focusing and zooming.



Public Variables

trackPinchGestures Whether pinch gestures should be tracked for zooming.
trackTapGestures Whether tap gestures should be detected for focusing.
ZoomDelta Returns the current zoom delta from pinch gesture tracking.
