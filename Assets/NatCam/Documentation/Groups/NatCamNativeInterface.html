NatCamNativeInterface



Description

The native interface for handling all camera operations. Do not use this directly if you don't have to, instead use NatCam.



Static Variables

FirstFrameReceived Returns whether the first frame of the preview has been received.
Preview The camera preview as a Texture2D.


Events

OnNativePreviewUpdate An event fired when NatCam receives the native pixel buffer handle for the preview frame.
