PreviewType

enumeration in NatCamU / Enumeration

Description

This represents whether the camera preview is readable or not. This means that the preview pixel data is available in system memory. This removes the need for very expensive and slow readbacks from the GPU.

If you only need to display the preview, then NonReadable would suffice. But if you require access to texture data for any manipulations or calculations (like OpenCV or AR), then use Readable.



Static Variables

NonReadable Preview will be kept stored VRAM and will not be accessible to the CPU.
Readable Preview will be made available in RAM and will be accessible to the CPU.
