NatCamPreviewScaler

class in NatCamU / inherits from NatCamPreviewBehaviour

Description

This component is useful for scaling UI Panels so that the camera preview projected onto them is not stretched when displayed.



Public Functions

Apply Scale the UI Panel to conform to a texture.
Apply Scale the UI Panel to conform to a custom size.


Public Variables

positionAtScreenCentre If true, the UI panel will be positioned at the centre of the screen.
scaleMode How the UI Panel will be scaled.
