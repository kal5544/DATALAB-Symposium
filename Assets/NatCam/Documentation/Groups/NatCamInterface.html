NatCamInterface

enumeration in NatCamU / Enumeration

Description

The interface on which NatCam's backend will be initialized. The native interface is the default. It supports iOS and Android, and provides access to the full featureset of NatCam. The fallback interface is a wrapper around the WebCamTexture API for platforms that do not support NatCam natively. If a device does not support the native interface, NatCam will automatically fallback to the fallback interface.



Static Variables

FallbackInterface The fallback interface (WebCamTexture API).
NativeInterface The native interface. Default.
