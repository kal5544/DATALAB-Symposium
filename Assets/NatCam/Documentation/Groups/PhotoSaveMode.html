PhotoSaveMode

enumeration in NatCamU / Enumeration

Description

This represents whether to save a captured photo to the device, and where to save it to.



Static Variables

DoNotSave Do not save any captured photos to the device gallery.
SaveToAppAlbum Save any captured photos to an album titled [app name] in the device gallery.
SaveToPhotos Save any captured photos to the device gallery.
