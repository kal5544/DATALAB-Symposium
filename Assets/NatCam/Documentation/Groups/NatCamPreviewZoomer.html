NatCamPreviewZoomer

class in NatCamU / inherits from NatCamPreviewBehaviour

Description

This component is useful for zooming the camera.



Public Variables

zoomMode This dictates how NatCam applies zooming.
zoomRatio This is the percent zoom ratio for the active camera.
zoomSpeed How fast NatCam's pinch-to-zoom will happen.
