NatCam



Description

This is the central class that receives all commands. All of NatCam's functionality is accessed through this class. NatCam is built for iOS and Android, but has a fallback API wrapper for the WebCamTexture API. Thus, you can use the same code for all the platforms you will be supporting.



Static Functions

CapturePhoto Capture a photo and send it back to the provided callbacks.
ExecuteOnPreviewStart Execute the PreviewCallbacks when the preview starts.
Initialize Initialize NatCam.
Pause Pause the camera preview.
Play Start the camera preview.
Release Stop the camera preview and release NatCam.
RequestBarcode Request to be notified when NatCam detects a barcode.
RequestFace Request to be notified when NatCam detects a face.
SaveToPhotos Save a photo to the camera roll.


Static Variables

ActiveCamera Get or set the active camera.
HasPermissions Returns whether NatCam has permissions to use the device cameras.
Interface Get the backend interface NatCam was initialized with.
IsPlaying Returns whether a camera preview is playing.
IsSupportedDevice Returns whether NatCam is running on a supported device.
PhotoSaveMode Get or set the photo save mode for subsequent captured photos.
Preview The camera preview as a Texture.
PreviewFrame The current camera preview frame.
PreviewMatrix The camera preview as an OpenCV Matrix.
Verbose Get or set NatCam's verbose logging mode.


Events

OnBarcodeDetect An event fired when NatCam detects a barcode.
OnFaceDetect An event fired when NatCam detects a face.
OnPhotoCapture An event fired when NatCam receives a captured photo.
OnPreviewStart An event fired when the camera preview has started.
OnPreviewUpdate An event fired when the camera preview updates.
