ComponentBuffer

enumeration in NatCamU / Enumeration

Description

The type of buffer (or hardware texture) which NatCam's OnNativePreviewUpdate broadcasts.

For textures (GPU pointers), the returned pointer will be a Texture Name (GLuint) on OpenGLES and an id<MTLTexture> on Metal.



Static Variables

RGBA32 The RGBA32 pixel buffer of the preview.
RGBA32GPU The RGBA32 preview texture on the GPU.
