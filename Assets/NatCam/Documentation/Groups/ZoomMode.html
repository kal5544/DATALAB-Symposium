ZoomMode

enumeration in NatCamU / Enumeration

Description

This handles how NatCam will zoom the active camera.



Static Variables

DigitalZoomAsFallback Use shader-accelerated digital zoom if hardware optical zoom is unavailable.
ForceDigitalZoomOnly Use shader-accelerated digital zoom irrespective of hardware optical zoom support.
ZoomSpeedOverrideOnly Only override NatCam's default pinch-to-zoom zoom speed.
