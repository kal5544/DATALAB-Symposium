UnitygramBase

class in NatCamU / inherits from MonoBehaviour

Description

A very useful MonoBehaviour class that can be used to quickly create NatCam client components. This class handles NatCam's initialization. It is also very easily subclassed and overriden to create custom and extended functionality.



Public Functions

OnPreviewStart Method called when the camera preview starts.
OnPreviewUpdate Method called on every frame that the camera preview updates.
Start The Start method of this MonoBehaviour called by Unity.
SwitchCamera Switch the active camera of the preview.


Public Variables

DetectMetadata Whether NatCam would be initialized with metadata detection.
Facing The facing of the camera NatCam will start the preview from.
FocusMode The focus mode that the camera will start with.
Framerate The framerate preset that the camera preview will have.
Interface The backing interface NatCam will be initialized with.
PreviewType The preview type.
RawImage The RawImage that will be the camera preview.
Resolution The resolution preset that the camera preview will have.
VerboseMode Whether NatCam should be verbose for debugging.
