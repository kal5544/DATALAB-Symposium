NatCamFallbackInterface



Description

The fallback interface for handling camera operations. This uses the Unity WebCamTexture API and as a result, possesses very limited support for the native NatCam features. Do not use this directly if you don't have to, instead use NatCam.



Static Variables

FirstFrameReceived Returns whether the first frame of the preview has been received.
Preview The camera preview as a WebCamTexture.
PreviewBuffer Get the Color32 buffer for the camera preview.
