NatCam.PreviewFrame

public static Texture2D PreviewFrame;

Description

The current camera preview frame. This returns a Texture2D. This will return null when running a NonReadable preview on the NativeInterface.

Note that this will allocate a new Texture2D each time it is called. To reuse an already-allocated Texture2D instead, comment '#define ALLOCATE_NEW_FRAME_TEXTURES' in NatCamNativeInterface.cs and NatCamFallbackInterface.cs.
