NatCam.ExecuteOnPreviewStart

public static void ExecuteOnPreviewStart (PreviewCallback[] callbacks);

Description

Execute the PreviewCallbacks when the preview starts. This is usually used to set a graphic's texture to the NatCam Preview. This step is necessary because prior to the OnPreviewStart event firing, NatCam.Preview is null. This is also used when setting things like exposure mode, focus mode, and similar on the active camera. This will also invoke callbacks when the active camera is switched.

This is equivalent to subscribing the supplied callbacks to the OnPreviewStart event. The difference is that the callbacks will automatically be unsubscribed once they have been invoked.
 

//Set our preview UI Panel RawImage texture
NatCam.ExecuteOnPreviewStart(() => rawImage.texture = NatCam.Preview);
