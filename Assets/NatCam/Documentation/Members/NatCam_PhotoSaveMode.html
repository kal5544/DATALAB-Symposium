NatCam.PhotoSaveMode

public static PhotoSaveMode PhotoSaveMode;

Description

Get or set the photo save mode for subsequent captured photos. The default is PhotoSaveMode.DoNotSave.

Note that on Android, setting this requires the 'External' Write Access permission in PlayerSettings. On the fallback interface, this is unsupported.

See Also: PhotoSaveMode .
