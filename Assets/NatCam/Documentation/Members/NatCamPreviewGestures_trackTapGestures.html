NatCamPreviewGestures.trackTapGestures

public bool trackTapGestures;

Description

Detect tap gestures for focusing. When set to true, NatCam will track touch gestures on the screen and apply focus accordingly.

Note that NatCam will only attempt to track touch gestures if the active camera's focus mode is either TapToFocus or HybridFocus.
