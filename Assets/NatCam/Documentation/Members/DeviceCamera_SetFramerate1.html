DeviceCamera.SetFramerate

public void SetFramerate (float framerate);

Description

Set the framerate of the camera to a supported framerate closest to (framerate). If the camera does not support the supplied framerate, the call will silently fail.

It is advised to set resolution before setting framerate because the supported framerates might be dependent on the active resolution on some platforms. 

//Set the rear camera's preferred framerate to 30FPS //For versatility, you are advised to use the FrameratePreset overload
DeviceCamera.RearCamera.SetFramerate(30f);
