DeviceCamera.ZoomRatio

public float ZoomRatio;

Description

Get or set the active camera's zoom ratio.

When setting this property, this will zoom the camera to a zoomRatio between no zoom and maximum zoom. Zoom ratio must be a value between [0,1]. NatCam will automatically clamp the value instead of throwing an ArgumentOutOfRangeException.

Keep in mind that this is a null op on device cameras that do not support optical zoom. For versatility in usage, you best use NatCamPreviewZoomer with ZoomMode.DigitalZoomAsFallback.

Also note that this is a null op if you have an active NatCamPreviewZoomer with ZoomMode.ForceDigitalZoomOnly.
