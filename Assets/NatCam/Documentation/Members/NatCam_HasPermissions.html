NatCam.HasPermissions

public static bool HasPermissions;

Description

Returns whether NatCam has permissions to use the device cameras. This must be true for NatCam to work properly. If it is false and NatCam.Initialize() or NatCam.Play() is called, the app could crash.

It is advised to use a coroutine to wait for a while after app launch before checking if the user has granted permissions and continuing. See the example below.
 

// Use this for initialization
IEnumerator Start () {
    //Wait for 5 seconds
    yield return new WaitForSeconds(5f);
    //Check if NatCam has the permissions
    if (NatCam.HasPermissions) {
        //Start NatCam
        StartNatCam();
    }
    //NatCam was not granted the permissions
    else {
        //Log an error
        Debug.LogError ("Don't use a camera app that can't use the camera! >:(");
    }
}
