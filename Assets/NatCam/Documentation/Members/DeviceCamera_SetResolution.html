DeviceCamera.SetResolution

public void SetResolution (ResolutionPreset preset);

Description

Set the preview resolution to one that satisfies the supplied resolution preset. 

//Set the front camera's preview resolution to medium resolution
DeviceCamera.FrontCamera.SetResolution(ResolutionPreset.MediumResolution);

See Also: ResolutionPreset .
