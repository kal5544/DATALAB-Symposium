NatCam.RequestFace

public static void RequestFace (FaceCallback[] callbacks);

Description

Request to be notified when NatCam detects a face. Note that when NatCam.Initialize is called, MetadataDetection must be on. It is off by default.

This is not supported on the Fallback Interface. 

void Start () {
    //Initialize NatCam with metadata detection
    NatCam.Initialize(MetadataDetection:Switch.On);
    //[Other initialization code...]
    //Start the camera preview
    NatCam.Play();
    //Request faces
    NatCam.RequestFace(OnDetectedFace);
}

void OnDetectedFace (Face face) {
    //Log what we found
    Debug.Log("Found a pretty face at " + face.position.ToString());
}
