NatCamPreviewGestures.trackPinchGestures

public bool trackPinchGestures;

Description

Detect pinch gestures for zooming. When set to true, NatCam will track pinch gestures on the screen and apply zooming.

Note that NatCam will only attempt to apply hardware optical zoom, which most devices don't support. As a result, it is advised to use the NatCamPreviewZoomer component which has the ability to fallback to shader-accelerated digital zoom. To change the speed of NatCam's pinch to zoom speed, add a NatCamPreviewZoomer component and change the zoomSpeed there; this will override NatCam's default pinch-to-zoom speed.
