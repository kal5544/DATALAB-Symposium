DeviceCamera.SetPhotoResolution

public void SetPhotoResolution (int width, int height);

Description

Set the photo resolution to a supported resolution closest to (width, height). 

//Set the active camera's preferred photo resolution to 3264x2448 //For versatility, you are advised to use the ResolutionPreset overload
NatCam.ActiveCamera.SetPhotoResolution(3264, 2448);
