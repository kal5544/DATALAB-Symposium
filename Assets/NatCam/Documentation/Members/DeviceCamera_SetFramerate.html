DeviceCamera.SetFramerate

public void SetFramerate (FrameratePreset preset);

Description

Set the framerate of the camera to one that satisfies the supplied framerate preset. If the camera does not support the supplied framerate, the call will silently fail.

It is advised to set resolution before setting framerate because the supported framerates might be dependent on the active resolution on some platforms. 

//Set the active camera's preferred framerate to smooth (60FPS)
NatCam.ActiveCamera.SetFramerate(FrameratePreset.Smooth);
