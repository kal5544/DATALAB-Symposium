NatCam.Initialize

public static void Initialize (NatCamInterface NatCamInterface, PreviewType PreviewType, Switch MetadataDetection);

Description

Initialize NatCam. This method must be called before any other calls to NatCam or DeviceCamera.

Interface defines what backing interface NatCam will be initialized with. The NativeInterface is default, but if the device does not support it, NatCam will fallback to the FallbackInterface which is a WebCamTexture wrapper. The FallbackInterface has very limited feature support.

PreviewType defines whether the preview is readable or not. Applications that require accessing the preview data for any CPU operations (like GetPixels, SetPixels, EncodeToJPG, and so on) will require PreviewType.Readable. Applications like OpenCV will also require this. It is advised that if the preview will only be displayed and nothing else, leave it NonReadable. This is NonReadable by default.

If you wish to detect metadata (faces and barcodes), you must pass set MetadataDetection to On. This is false by default.
 

void Start () {
    //Initialize NatCam for barcode and face detection
    NatCam.Initialize(Interface:NatCamInterface.NativeInterface, MetadataDetection:Switch.On);
    //Play the rear camera
    NatCam.Play(DeviceCamera.RearCamera);
    //Set our preview UI Panel RawImage texture
    NatCam.ExecuteOnPreviewStart(() => rawImage.texture = NatCam.Preview);
}
    

void Start () {
    //Initialize NatCam with default settings
    NatCam.Initialize();
    //Play the rear camera
    NatCam.Play(DeviceCamera.RearCamera);
    //Set our preview UI Panel RawImage texture
    NatCam.ExecuteOnPreviewStart(() => rawImage.texture = NatCam.Preview);
}
